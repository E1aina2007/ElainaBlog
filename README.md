# ElainaWeb 基于gin和vue3+ts的个人小窝(网站)
边学边搭建, 搭到哪学到哪

## 技术栈
- **后端**: 
    - **框架**: Gin + GORM
    - **数据库**: MySQL
- **前端**: 
    - **框架**: Vue3 + TypeScript
    - **状态管理**: Pinia
    - **路由**: Vue-router
    - **构建工具**: Vite
- **服务器**: Nginx
- **部署**: Docker
